## Sample Metadata

- **Sample**: NA12878 (HG001) — the Genome in a Bottle reference sample
- **Library**: Illumina paired-end, ~75 bp reads (low-coverage 1000G run)
- **Coverage**: ~4-6x on chr20 (sufficient to learn the pipeline; production WGS uses 30x)
- **Reference build**: hg38 (GRCh38), UCSC chromosome naming (`chr20`)

## Why chr20?

chr20 (~63 Mb) is large enough to produce meaningful variant calls but small enough
to process in minutes on a laptop. All pipeline steps work identically on full-genome
data — just point `--reference` at the full hg38 FASTA.

## Note on Read Coordinates

The 1000 Genomes chr20 BAM was originally aligned to **GRCh37** (hg19). The download
script extracts raw reads as FASTQ (discarding the original alignment coordinates) and
the pipeline re-aligns them to **hg38**. Because chromosome 20 is highly conserved
between builds, the reads map cleanly to hg38 — this has no effect on learning the
pipeline mechanics.
